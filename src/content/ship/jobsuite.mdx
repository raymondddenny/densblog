## The Tech Behind It

Built with a modern stack focused on speed and reliability:

- **Supabase** for authentication and database
- **Google Sign-In** for seamless onboarding
- **OpenRouter** for AI content generation
- **Cloudflare** for security and bot protection
- **Polar** for payment processing
- **Fly.io** for hosting

Security was a priority ‚Äî all data is encrypted in transit (HTTPS) and at rest via Supabase.
